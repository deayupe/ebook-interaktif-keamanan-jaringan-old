Keamanan Akun Sistem Linux










                     root:x:0:0:root:/root:/bin/bash






                 Setiap  string  dalam  baris  tersebut  dipisahkan  oleh  simbol  ":"

                 yang masing-masing memiliki arti sebagai berikut.


                     1. root  adalah  nama  akun  yang  digunakan  untuk  login  ke


                        terminal Linux.

                    2.  x  mereferensikan  simbol  tentang  enkripsi  password


                        pengguna root yang tersimpan dalam /etc/shadow.


                    3.  Angka  0  menjelaskan  nomor  UID  untuk  superuser  dengan


                        akun  root.  Penomoran  UID  digunakan  untuk  memudahkan

                        sistem Linux mengidenfikasi proses yang sedang dikerjakan


                        oleh pengguna tertentu. Kapasitas penomoran UID memiliki


                        jangkauan  16-bit  integer  dari  0  sampai  65.535.  Nilai  UID


                        antara  0  sampai  9  secara  khusus  digunakan  pengguna

                        yang  berfungsi  menjalankan  layanan  sistem,  sedangkan


                        nomor  20  atau  100  digunakan  untuk  nomor  UID  pada


                        pengguna reguler.


                    4.  Angka 0 berikutnya mereferensikan nomor GID ketika akun

                        tersebut  tergabung.  Bilangan  0  memiliki  arti  bahwa  grup


                        tersebut  merupakan  grup  pengguna  yang  memiliki


                        kewenangan tanpa batas atau superuser.


                    5.  root  berikutnya  menunjukkan  penjelasan  tentang  detail

                        atau nama lengkap akun root.


                    6.  /root  menjelaskan  direktori  kerja  yang  dimiliki  oleh  akun


                        root ketika login.


                    7.  /bin/bash menunjukkan jenis shell yang digunakan secara

                        defoult oleh akun root.



                 b. Regular account

                     Jenis  akun  ini  merupakan  akun  yang  ditambahkan  secara


                 manual  sesuai  kebutuhan  akses  terhadap  sistem  Linux.  Tipikal

                 akun  ini  memiliki  akses  dan  kontrol  terbatas  terhadap  sistem,


                 tetapi  dapat  memperoleh  hak  istimewa  setara  root  dengan

                 menggunakan  perintah  sudo  atau  su.  Dalam  sistem  operasi

                 Linux  dengan  arsitektur  modern,  pengguna  biasa  memiliki  UID


                 dengan  komposisi  empat  digit  yang  dimulai  dari  1.000.  Untuk

                 direktori  kerja,  akun  pengguna  regular  mempunyai  direktori


                 default yang berada di dalam direktori /home. Sebagai contoh,


                 pengguna "andi" akan terbentuk direktori kerjanya di /home/

                 andi. Selain itu, pengguna ini juga telah ditentukan shell


                 default yang umumnya menggunakan Bourne-Again Shell


                 (Bash). Anda dapat mengubah jenis shell default-nya

                 menjadi shell C atau shell Z.
