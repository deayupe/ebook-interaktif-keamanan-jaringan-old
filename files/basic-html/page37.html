4) Tingkat keamanan lebih tinggi

                      Dengan melacak status koneksi, firewall stateful dapat mencegah

                      serangan  yang  mencoba  mengeksportasi  protokol  jaringan  atau


                      menyamar sebagai lalu lintas yang sah seperti serangan spoofing,

                      pemalsuan                    sesi          TCP,          atau            serangan                 yang             mencoba


                      memanfaatkan  koneksi  yang  sudah  ada  untuk  menyusup  ke

                 5) Inspeksi paket tebih mendalam

                      Firewall stateful juga dilengkapi dengan fitur inspeksi paket secara

                      mendalam, yaitu filtering isi data paket untu mendeteksi malware,


                      virus, atau ancaman keamanan lainnya.

        2. Proxy Service Firewall


            Firewall  proxy  berperan  sebagai  perantara  antara  sistem  pengguna

            (klien)  dan  sumber  daya  yang  mereka  akses  di  internet  (server).

            Artinya, semua permintaan dari pengguna ke internet dan sebaliknya


            harus melewati firewall ini. Teknologi firewall ini mampu bekerja hingga


            lapisan  aplikasi  sehingga  dapat  menyaring  lalu  lintas  jaringan

            berdasarkan konten aplikasi yang digunakan, seperti protokol lapisan 7

            (HTTP, HTTPS, FTP, SFTP, dan lainnya).


            Cara kerja firewall proxy adalah sebagai berikut.





            a. Sebagai perantara


                 Firewall  proxy  bertindak  sebagai  gateway  antara  klien  dan  server.

                 Ketika  klien  membuat  permintaan  ke  server  di  internet,  permintaan


                 tersebut  akan  dikirim  ke  firewall  proxy  terlebih  dahulu,  selanjutnya


                 proxy  membuat  permintaan  baru  ke  server  yang  dituju  atas  nama

                 klien.  Server  tujuan  hanya  berinteraksi  dengan  firewall  proxy  yang

                 mampu  menyembunyikan  informasi  detail  klien  (contoh  alamat  IP)


                 guna  meningkatkan  keamanan  dan  anonimitas  klien.  Proxy  juga

                 dapat melakukan caching, yaitu menyimpan salinan sementara dari


                 konten yang sering diakses guna mempercepat waktu respons dan

                 mengurangi beban jaringan.





            b. Evaluasi keamanan


                 Sebelum meneruskan permintaan (request) baik dari klien ke server


                 maupun  respon  dari  server  ke  klien,  firewall  proxy  melakukan

                 evaluasi lalu lintas berdasarkan serangkaian aturan keamanan yang


                 telah ditentukan. Evaluasi ini mencakup lima hal berikut.


                        Proxy  mengidentifikasi  malware  dengan  memindai  lalu  lintas

                        untuk  mencari  tanda-tanda  malware  dan  mencegahnya


                        mencapai klien atau server.
